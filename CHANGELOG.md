# Changelog

## [2.0.0] - 2026-05-10
### Added
- **Interactive TUI**: New full-screen dashboard using Bubble Tea.
- **API-less AI**: Switched to public AI model endpoints (no API key required).
- **Clipboard Support**: `Ctrl+Y` to copy suggested commands.
- **Direct Execution**: `Ctrl+R` to run commands with confirmation.
- **Safety Engine**: Real-time analysis to block or warn against dangerous commands.
- **Command History**: `syniq history` to review past queries.
- **Version Flag**: Added `syniq version` and `-v/--version` support.

### Changed
- Improved CLI aesthetics with Lip Gloss.
- Updated README with new features and usage guide.
- Enhanced "Thinking" animation in CLI mode.

### Fixed
- Fixed fuzzy matching in local cache.
- Resolved issues with quote handling in `ask` command.
